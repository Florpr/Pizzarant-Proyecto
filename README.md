Proyecto Full-Stack: Sistema de Gestión Pizzarant

Descripción

Aplicación web completa que permite la gestión de pedidos, menú, inventario y usuarios para un restaurante de pizza, incluyendo autenticación, operaciones CRUD (Crear, Leer, Actualizar, Borrar) y despliegue.


Node.js

PostgreSQL

React/Angular/Vue

Instalación

git clone <repo>
cd server
npm install
cd ../client
npm install


Ejecución

# server
npm start
# client
npm start
